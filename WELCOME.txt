📱 ЧТО ЭТО?
───────────────────────────────────────────────────────────────────────────

IBO Phone ISO - это мобильная операционная система на базе Linux с поддержкой:

  ✓ IBO App Store       - Магазин приложений для установки и управления
  ✓ IBO Browser         - Веб-браузер для просмотра интернета
  ✓ APK и AAB пакеты   - Поддержка всех форматов приложений Android
  ✓ Linux Kernel 5.15   - Современное ядро с SELinux безопасностью
  ✓ Docker поддержка    - Легко развернуть в контейнере
  ✓ 2590+ строк кода    - Полностью функциональная система


🚀 БЫСТРЫЙ СТАРТ (3 способа)
───────────────────────────────────────────────────────────────────────────

1️⃣  ИНТЕРАКТИВНЫЙ ИНСТАЛЛЯТОР (РЕКОМЕНДУЕТСЯ)
    
    bash install.sh
    
    Затем выберите в меню:
    - 2) Подготовить систему
    - 3) Построить ISO образ

    bash build/setup-vm.sh && bash build/build.sh
    
    Результат: build_output/ibo-phone-1.0.0.iso

3️⃣  DOCKER (В КОНТЕЙНЕРЕ)
    
    docker build -f build/Dockerfile -t ibo-phone:latest .
    docker run -it ibo-phone:latest bash

🔧 ОСНОВНЫЕ КОМАНДЫ
───────────────────────────────────────────────────────────────────────────

Установка приложений:
  bash system/install_packages.sh install-apk app.apk com.example.app
  bash system/install_packages.sh install-aab app.aab com.example.app

Управление приложениями:
  bash system/install_packages.sh list          # Список установленных
  bash system/install_packages.sh uninstall ... # Удалить приложение

Сборка системы:
  bash build/setup-vm.sh                        # Подготовка ВМ
  bash build/build.sh                           # Сборка ISO образа

Интерактивный режим:
  bash install.sh                               # Главное меню

📦 ПОДДЕРЖИВАЕМЫЕ ФОРМАТЫ
───────────────────────────────────────────────────────────────────────────

APK (Android Package)
  - Стандартный формат приложений Android
  - Размер: до 500MB
  - Формат: ZIP архив
  - Поддержка: все версии Android

AAB (Android App Bundle)
  - Новый формат от Google
  - Оптимизированный размер
  - Требует bundletool для установки
  - Рекомендуется для больших приложений

🚢 РАЗВЕРТЫВАНИЕ
───────────────────────────────────────────────────────────────────────────

На виртуальной машине:
  1. Собрать ISO: bash build/build.sh
  2. Загрузить ISO в VM (QEMU, VirtualBox)
  3. Установить на VM
  4. Запустить систему

В Docker контейнере:
  1. Собрать образ: docker build -f build/Dockerfile -t ibo-phone .
  2. Запустить контейнер: docker run -it ibo-phone:latest bash
  3. Собрать ISO внутри контейнера: bash build/build.sh

На облачных платформах:
  1. Создать QEMU образ
  2. Загрузить на AWS/GCP/Azure
  3. Запустить виртуальный сервер
