2. Или шаги вручную:
   $ bash build/setup-vm.sh        # Подготовка

   $ bash system/install_packages.sh install-apk app.apk com.example.app

3. Docker:
   $ docker build -f build/Dockerfile -t ibo-phone:latest .
   $ docker run -it ibo-phone:latest bash
